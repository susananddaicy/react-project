# Swiper组件
基于Swiper的React版本:
http://idangero.us/swiper/#.WAcXTZN95TY

支持所有Swiper的option和API:
http://idangero.us/swiper/api/#.WAc1SZN95TY

options:
```javascript
    let swipeOptions = {
      initialSlide: 1,
      slidesPerView: 3,
      touchRatio: 0.4,
      resistanceRatio: 0.4,
      centeredSlides: true,
      onSlideChangeEnd: this.changeSwiperState,
    };
```
